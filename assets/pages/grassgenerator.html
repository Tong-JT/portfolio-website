Autodesk Maya Grass Generator

This tool for allows users to generate grass blades in a Maya scene. Users can customize grass length, density, and placement on selected surfaces.

This was a personal project to combine my interest in art and programming. Since Maya is an industry standard for creating 3D models, I wanted to experiment with creating plugins for it. Python was used to develop the entire plugin, utilising Qt6 (Pyside2) for the interface and vanilla Python for the logic.
